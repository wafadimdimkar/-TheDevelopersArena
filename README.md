Step 1: Taking User Input<br>
The program begins by asking the user basic personal questions using the `input()` function. <br> 
Each answer provided by the user is stored in a variable.<br>

Example:<br>
- Name is stored in a variable<br>
- Age is stored in a variable<br>
- Hobby is stored in a variable<br>

This demonstrates how Python interacts with users and stores data in memory.<br>

 Step 2: Using Variables<br>
Variables are used to store user-provided data such as name, age, and hobby.<br>  
These variables allow the program to reuse the information later while displaying output.<br>

 Step 3: Displaying Output<br>
The program uses `print()` along with **f-strings** to display a friendly and personalized welcome message.<br>  
F-strings help insert variable values directly into text, making the output clean and readable.<br>

 Step 4: Program Flow<br>
The program executes sequentially:<br>
1. Ask questions<br>
2. Store responses<br>
3. Display personalized message<br>

This helps in understanding how Python runs code from top to bottom.<br>
