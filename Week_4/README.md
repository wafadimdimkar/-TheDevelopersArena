 Week 4: Data Visualization & Complete Data Analysis Project<br>
 Project Overview<br>
This project is part of **Week 4 – Data Visualization**.  <br>
The objective is to perform a complete data analysis workflow using Python, including data loading, cleaning, analysis, and visualization.<br>

The project analyzes an e-commerce sales dataset and presents insights using charts and written explanations.<br>

 Project Objectives<br>
- Understand why data visualization is important<br>
- Perform end-to-end data analysis<br>
- Create meaningful charts using matplotlib<br>
- Present insights clearly and professionally<br>
- Build a complete, well-documented project<br>

 Technologies Used<br>
- Python 3<br>
- pandas<br>
- matplotlib<br>
- Google Colab / Jupyter Notebook<br>
